# Hire Flow — Automated Resume Screening & Interview Scheduling

Monorepo structure:

- `frontend/`: React (Vite) + Tailwind + UI components (Phase 1)
- `backend/`: Node.js + Express (Phase 2)

## Phase 1 (Frontend) — run locally

Prereqs: Node.js 18+

```bash
cd frontend
npm install
npm run dev
```

Then open the URL shown in the terminal.
